## О проекте

Чистая, доступная страница авторизации: анимированный фон-аврора, стеклянная карточка, встроенная валидация, заготовки для OAuth и полированные состояния загрузки и успеха — всё на чистом HTML, CSS и JS без зависимостей. Сборка не требуется.

---

## Возможности

| Фича | Подробности |
|---|---|
| Статичные лейблы | Чёткие подписи над каждым полем — лучше читаются |
| Иконки в полях | Конверт для email и замок для пароля — мгновенный контекст |
| Показ пароля | Иконка глаза с ARIA-состоянием `pressed`; фокус возвращается в поле |
| Забыли пароль? | Прямо рядом с лейблом поля — легко найти |
| Живая валидация | Ошибки при потере фокуса, исчезают как только ввод стал верным |
| Алерт ошибки | Анимированная плашка при неверном логине; перезапускает анимацию при повторной ошибке |
| Состояние загрузки | Кнопка блокируется и показывает спиннер во время запроса |
| Состояние успеха | Кнопка зеленеет с галочкой при успешном входе |
| OAuth-кнопки | Заготовки для Google и GitHub — подключи свой SDK |
| Фон-аврора | Три анимированных радиальных градиента + тонкая сетка |
| Анимация карточки | Плавное появление при загрузке страницы |
| Доступность | `role="alert"`, `aria-live`, `aria-pressed`, обводки `focus-visible` везде |
| Адаптивность | Работает от 320px; OAuth-кнопки складываются в столбик на мобильных |
| Без анимаций | Все эффекты отключаются при `prefers-reduced-motion: reduce` |

## Смена темы

Все цвета — CSS-переменные в `:root`:

```css
:root {
    --accent:     #818cf8;   /* индиго — обводки фокуса, ссылки, чекбокс */
    --accent-alt: #c084fc;   /* фиолетовый — конец градиента */
    --bg:         #080c1a;   /* тёмно-синий фон */
    --danger:     #f87171;   /* ошибки валидации */
}
```

Два изменения для ребрендинга: поменяйте `--accent` + `--accent-alt` и обновите градиент `.btn-submit`.

## Доступность

- Все интерактивные элементы доступны с клавиатуры, обводки `focus-visible` везде
- Ошибки используют `role="alert"` и `aria-live="polite"` — скринридеры их озвучивают
- Переключатель пароля динамически обновляет `aria-pressed` и `aria-label`
- Анимации авроры и карточки отключаются при `prefers-reduced-motion: reduce`
- Чекбокс использует скрытый нативный `<input>` для полной поддержки клавиатуры и скринридеров
